# Guideline

> Installation d'un émulateur Mini vMac 

→ vous aurez besoin de votre ordinateur et d'une connexion internet (c'est tout !)



1. **Créez un dossier** sur votre ordinateur pour garder tout bien organisé 
   
       ex : `MinivMac` 

2. **Téléchargez** le mini vMac sur votre ordinateur (attention à installer en fonction, Mac, Windows, linux !!!)
   
     → [Download Mini vMac 36.04 Standard Variations](https://minivmac.github.io/gryphel-mirror/c/minivmac/dnld_std.html) 
   
     *dé-zippez le document et collez le minivMac dans votre dossier `MinivMac`*

3. **Téléchargez une ROM** Macintosh Plus (128 KB)
   
    → [Macintosh 128k - ROMs MESS - MESS - ROMs - Planet Emulation](https://www.planetemu.net/rom/mess-roms/macintosh-128k) 
   
   *Placez le dans votre dossier `MinivMac`*

    

4. **Lancez Mini vMac**
   
   - Double-cliquez = une icône disquette s'affiche 
   
   

5. **Téléchargez** une image disque 
   
    → [Mac System 6.0.8 for MiniVMac - Macintosh Repository](https://www.macintoshrepository.org/16994-mac-system-6-0-8-for-minivmac) 
   
   *Téléchargez le premier lien 6.08_40MB.img*
   
   *Déplacez-là dans votre dossier `MinivMac`*
   
   

6. **Glisser** (drag and drop) une image dans le Mini vMac
   
   - Drag and drop l'image disque dans l'interface du Mini vMac
   
   - fini!!!!
